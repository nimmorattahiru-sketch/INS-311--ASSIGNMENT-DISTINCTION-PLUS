# INS311-ASSIGNMENT-DISTINCTION-PLUS

## Overview

This is my **INS311 E-commerce Store project**. The website is built using **HTML, CSS, and VS Code**.  
It demonstrates modern web development practices and a **responsive design** that works on desktop, tablet, and mobile devices.

## Features

- **Three pages:** Home, Shop, and Blog
- **Three posts** with at least **two comments**
- **Three media files** (images for products and banner)
- **Featured products section** with interactive buttons
- **Navigation menu** and **social media link**
